# Sandbagger Invitational Starter Website

Official starter site for baggerinv.com.

## Upload to GitHub

1. Open the `BaggerInv` repository.
2. Click **Add file** → **Upload files**.
3. Upload every file and folder from this package.
4. Commit the changes to the `main` branch.
5. Vercel should automatically redeploy.

## Run locally (optional)

```bash
npm install
npm run dev
```

Then open http://localhost:3000.
